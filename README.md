# Osung Groupware

_오성 그룹웨어 프로젝트_

## 소개

- 회사 내 일정, 전자결재, 사내 메일, 근태 관리 등을 한번에 처리해 업무 효율을 높일 수 있게 해주는 그룹웨어 서비스
- 개발기간: 2020.11.24 ~ 2021.01.03 (약 4주)
- 개발인원: 5명

## 기술스택

> Back-end/Server

- Java
- Spring Framework MVC
- Jsp
- Java Servlet
- Apache Tomcat v8.5
- OracleDB
- Mybatis 

> Front-end

- Javascript
- JQuery
- HTML 5 & CSS3
- Bootstrap 3

> Devops

- Maven
- Git
- Github

## 기능

본인이 구현한 기능은 **굵게** 강조했습니다.

* 쪽지
  * 쪽지쓰기
  * 보낸쪽지함
  * 받은 쪽지함
  * 중요쪽지함
  * 임시보관함
  * 예약보관함
  * 예약쪽지함
  * 휴지통
* 전자결재
  * 새결재진행
  * 결재대기중
  * 진행중문서
  * 완료문서
* 근태관리
  * 연차신청
  * 승인대기중
  * 진행중문서
  * 완료문서
  * 출퇴근현황
* **ToDo+**
* **일정**
  * **캘린더**
  * **내 캘린더 관리**
* **예약**
  * **예약하기**
  * **나의 예약 목록**
